# BMW CAN Sender v3.3 Preserve+UI Upgrade

这个版本基于你上传的原始多文件版本做增强，目标是：

- 保留原来的开机画面、主菜单、发送格式选择、模式 A/B/C/D、热键、日志、历史、编辑器、Mode C 管理 GUI
- 不删除任何原有功能和文本
- 只做加法：补一个更符合人逻辑的会话控制中心，并把你要求新增的几个 ID 放进固定在线帧体系和编辑器

## 新增内容

- `U` 打开新增 **会话控制中心**（进入模式后也可自动打开）
- 设置中心新增：`自动打开会话控制中心`
- 新增固定在线帧条目：
  - `0x0DF` 换挡提示
  - `0x2C4` 水温/油温表（先按 8 字节样本透传）
  - `0x349` 燃油
  - `0x36E` ABS
- 保留原来的终端流程和所有按键

## 运行

```bash
python main.py
```

## 说明

- 这个版本不是重写，不是替换原逻辑，而是在原逻辑上叠加 UI。
- 你原来的命令行和窗口仍然都在。
- 新增会话控制中心只是补一个更现代、更集中、更顺手的面板。
